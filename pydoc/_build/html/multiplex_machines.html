cross_over(other)¶

Performs a crossing-over between two chromosomes: self and other. if the cutting point < 12 (half of length), exchange components from the front end part of the two chromosomes if cutting point > 12, exchange components from the rear end part of the two chromosomes So if cutting point = 2, exchange the first 2 components. If cutting point = 22, exchange the three last components.

Parameters:other – a other instance of chromosome object
Returns:nothing but performs the cross-over
